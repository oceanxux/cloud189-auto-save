# Alist 本项目中的使用场景

Alist 是一款支持多种存储的目录列表程序。Cloud189 Auto Save 与 Alist 结合，可以极大扩展您的媒体管理能力。

---

## 1. STRM 文件生成

Alist 常用于挂载各种网盘。如果您希望为 Alist 挂载的所有内容生成 STRM 文件以便 Emby 使用：
- **云端路径前缀**：在 Alist 中访问该文件的基础路径（如 `/189Cloud`）。
- **本地路径前缀**：在本系统中生成 STRM 的目标路径（如 `/home/strm`）。

---

## 2. 播放直链换取

本项目内置的流代理可以通过 Alist 接口动态换取文件直链。
- **优点**：即使天翼云接口变动，只要 Alist 能正常播放，本项目生成的 STRM 就能正常播放。
- **配置**：在“系统设置” -> “Alist 配置”中填入 `Server URL` 和 `Token`。

---

## 3. 联动 Emby 播放

当您使用 Emby 播放 STRM 文件时：
1. Emby 发起播放请求。
2. 本项目收到请求，调用 Alist API 获取天翼云最新直链。
3. 本项目将直链返回给播放器，实现 4K 高码率流畅播放。

---

## 💡 使用小窍门
- **批量同步**：如果您在 Alist 中对 189 网盘的文件进行了手动重命名，可以使用本系统的“同步 Alist 结构”功能，一键更新本地所有 STRM 文件。
- **性能优化**：建议将 Alist 与本项目部署在同一局域网或同一宿主机内，以减少 API 调用的网络延迟。
